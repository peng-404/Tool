# Win-Deep-Cleaner

一个给普通 Windows 用户使用的清理辅助工具。

它不是“一键自动乱删”的脚本，而是分成两个阶段：

1. 先扫描电脑里的清理候选项，生成审查清单。
2. 再由你手动确认后，执行删除，并保留备份和报告。

## 这个工具是干什么的

Win-Deep-Cleaner 主要用来帮你找出电脑里比较常见、比较适合清理的内容，例如：

- 第三方启动项
- 常见应用缓存
- 开发环境缓存
- 下载、桌面、文档、临时目录中的旧安装包、压缩包和较大的久未使用文件
- 用户目录里的可疑可执行文件、自启动痕迹和脚本文件

它会把这些内容整理成一份可审查的列表，先让你看，再决定要不要删。

## 它的工作方式

### 阶段一：扫描并生成清单

点击 `Start Phase One` 后，程序会：

- 执行系统自带的基础清理
- 扫描常见缓存目录
- 扫描第三方启动项
- 递归扫描 `Downloads`、`Desktop`、`Documents`
- 递归扫描 `AppData\\Local\\Temp`
- 扫描部分可疑脚本、可执行文件和用户目录下的异常启动痕迹
- 生成两份文件到桌面：

`AI-Review-List.txt`

`AI-Review-Metadata.json`

其中：

- `AI-Review-List.txt` 是给你审查用的
- `AI-Review-Metadata.json` 是程序内部读取的元数据

### 阶段二：确认后执行

当你审查完 `AI-Review-List.txt` 后，点击 `Start Phase Two`，程序会：

- 重新读取你保留下来的项目
- 以勾选列表的方式展示出来
- 允许你再次取消勾选
- 支持先备份
- 支持优先移动到回收站
- 执行完成后生成 TXT 和 JSON 报告

## 怎么操作

1. 双击运行 [双击运行我.bat](./双击运行我.bat)
2. 在主界面点击 `Start Phase One`
3. 等待桌面生成 `AI-Review-List.txt`
4. 打开 `AI-Review-List.txt`
5. 删除你不想处理的行，只保留你准备继续检查或执行的项目
6. 保存这个 TXT 文件
7. 回到程序主界面，点击 `Start Phase Two`
8. 在弹出的勾选窗口中再次确认
9. 选择是否备份、是否优先移动到回收站
10. 点击执行

## 适合清理什么

- 浏览器缓存
- 微信和 QQ 的部分缓存或临时目录
- HuggingFace、PyTorch、pip、Conda、npm 等开发缓存
- 下载、桌面、文档里长期未使用的安装包和压缩包
- 临时目录中长期未使用的文件
- 用户安全区域内较大的旧文件
- 可疑启动项、可疑脚本、可疑可执行文件

## 当前扫描范围

程序现在不是全盘扫描，而是“准全盘扫描”，主要递归检查这些相对安全区域：

- `C:\\Users\\<用户名>\\Downloads`
- `C:\\Users\\<用户名>\\Desktop`
- `C:\\Users\\<用户名>\\Documents`
- `C:\\Users\\<用户名>\\AppData\\Local\\Temp`
- 可选的大文件目录，例如 `C:\\Users\\<用户名>\\Videos`
- 常见缓存目录
- 用户启动目录

另外还会补充扫描：

- 第三方启动项
- 用户目录里近期出现的可执行文件和脚本
- 名称伪装成系统进程的可执行文件
- 双扩展名可执行文件，例如 `xxx.pdf.exe`

## 不适合直接清理什么

- Windows 系统目录
- 驱动相关目录
- 你不确定用途的软件安装目录
- 个人文档、照片、工作资料
- 需要复杂依赖分析的开发环境本体

## 安全说明

程序内置了一些基础保护措施，例如：

- 高风险系统路径会被自动拦截
- 第二阶段执行前会再次检查路径
- 可以先备份
- 可以优先移动到回收站
- 执行后会生成报告
- 可疑项默认更适合先审查，不代表程序已经判定它一定是病毒

但这仍然是一个会实际删除文件的工具，所以请一定在第二阶段认真检查最终勾选内容。

## 注意事项

- 运行时通常需要管理员权限
- 建议先看清楚审查清单，再执行阶段二
- 如果你不确定某个路径能不能删，就不要删
- 对重要数据请提前自行备份
- “可疑项”只是启发式提示，不等于杀毒软件的查杀结论
