# Win-Deep-Cleaner (AI-Enhanced)

这是一个结合了**Windows底层原生优化**与**大语言模型 (LLM) 辅助决策**的深度系统清理与流氓软件排查工具。它能够穿透隐藏属性，精准打击隐形缓存与顽固驻留程序。

## ✨ 核心特性
- **底层原生清理**：自动调用 `dism` 和 `cleanmgr`，深度清理 WinSxS 组件库与系统更新残留。
- **全领域靶向扫描**：精准定位隐藏在系统深处的流氓自启项、国民级应用缓存（微信/QQ/浏览器）以及高频开发工具冗余（PyTorch/Npm 等）。
- **AI 辅助排查 (创新)**：脚本不盲目删文件，而是生成标准化清单，支持无缝对接各大 AI 模型进行逻辑排查。
- **三重防误删机制**：人工初筛 -> AI 逻辑诊断 -> 终极可视化弹窗确认。
- **穿透式物理查杀**：自动结束占用进程，无视“隐藏”与“只读”属性强制粉碎目标。

## 🚀 终极三步走工作流 (零基础友好)

**第一步：运行脚本**
1. 下载源码，右键点击 `Win-Deep-Cleaner.ps1`，选择 **“使用 PowerShell 运行”**。
2. 脚本会自动静默清理 Windows 垃圾，并扫描出电脑里的疑似流氓软件和隐形缓存，随后自动打开一个 `AI审查清单.txt` 记事本。

**第二步：让大模型替你诊断 (防误删核心)**
你不必懂代码，直接将记事本里的全部文字复制，发送给任何大模型（如 Kimi、文心一言、ChatGPT 等），并附上这句话：

> "我不懂电脑，请你帮我诊断这些 Windows 路径。
> 1. 保留微软系统核心文件以及 Intel/NVIDIA/AMD 等正规硬件驱动。
> 2. 帮我找出并**删除**那些臭名昭著的弹窗广告程序、流氓卫士、捆绑全家桶的自启文件。
> 3. 帮我**删除**清单里列出的应用缓存（如微信小程序缓存、浏览器缓存、临时文件）。
> 4. 请直接给我最终需要删除的纯路径列表，每行一个，不要任何解释废话。"

**第三步：终极查杀**
1. 将 AI 给出的路径**覆盖粘贴**回打开的记事本中。
2. **保存 (Ctrl+S) 并关闭记事本**。
3. 脚本会弹出一个**红色警告清单窗口**，显示所有即将被粉碎的路径。
4. 确认无误后点击执行，脚本将自动拔除进程并物理粉碎文件。
